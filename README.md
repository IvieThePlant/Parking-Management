# Parking Management

## Setup instructions

### Admin Side

To access admin side of application, first create an admin account for yourself by calling

```python
python manage.py createsuperuser
```

Enter in the login details you want for your admin account.

Then, start the app with

```python
python manage.py runserver 
```

If you visit [127.0.0.1:8000/admin/](http://127.0.0.1:8000/admin/), you will now be prompted to log-in with your admin account.
